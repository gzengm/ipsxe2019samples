チュートリアル: インテル® IPP 2018 による画像の回転とブラー

アプリケーションのビルド

注

ipp_blur_rotate.cpp は、インテル® Parallel Studio XE 内にインストールされるインテル® IPP 用に作成されたサンプルコードです。このサンプルコードは、インテル® System Studio のようなほかの製品スイート内でも利用できます (グラフィカル・レンダリング部分を除く)。

ビルド環境のセットアップ

インテル® IPP とともにインストールしたスイートのコンパイラー・コンポーネントを起動する前に、コンパイラーの関連コンポーネントの場所を定義する特定の環境変数を設定する必要があります。インテル® C++コンパイラーをインストールすると、環境変数を設定するための compilervars スクリプトがインストールされます。

  • Windows* では、compilervars.bat バッチファイルが <install-dir>\compilers_and_libraries_2018.x.xxx\windows\bin\ にインストールされます。
  • Linux* および macOS* では、compilervars.sh シェルスクリプトが <install-dir>\compilers_and_libraries_2018.x.xxx\<linux|mac>\bin\ にインストールされます。

異なる製品スイートの環境変数の設定については、https://software.intel.com/en-us/articles/setting-up-the-build-environment-for-using-intel-c-or-fortran-compilers (英語) を参照してください。

コードのコンパイルとリンク

このチュートリアルのコードサンプルをコンパイルしてリンクするには、次の操作を行います。

  • Windows*:
    1. インテル® ソフトウェア製品のサンプルとチュートリアル (英語) からダウンロードした ipp_blur_rotate_sample.zip アーカイブを展開します。
    2. Microsoft* Visual Studio* コマンドプロンプトを実行して、Makefile ディレクトリーに移動します。デフォルトでは、Makefile は ipp_blur_rotate_sample.zip アーカイブのルートにあります。
    3. nmake -f Makefile_win コマンドを実行します。
  • Linux* および macOS*:
    1. インテル® ソフトウェア製品のサンプルとチュートリアル (英語) からダウンロードした ipp_blur_rotate_sample.tgz アーカイブを展開します。
    2. bash シェルを実行して Makefile ディレクトリーに移動します。デフォルトでは、Makefile は ipp_blur_rotate_sample.tgz アーカイブのルートにあります。
    3. make -f Makefile_lin コマンド (Linux*) または make -f Makefile_mac コマンド (macOS*) を実行します。

アプリケーションを実行するには、Makefile があるディレクトリーから次のコマンドを実行します。

  • Windows*: ipp_blur_rotate.exe
  • Linux*: ./ipp_blur_rotate
  • macOS*: ./ipp_blur_rotate.app/Contents/MacOS/ipp_blur_rotate
