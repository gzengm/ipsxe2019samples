Intel® Integrated Performance Primitives Image Blurring and Rotation Sample

The Image Blurring and Rotation sample is designed to illustrate how to use Intel® Integrated Performance Primitives (Intel® IPP) functions for image affine warping and filtering.

System Requirements:
  • A system based on one of the following processor types:
    • Intel® Core™ processors (2nd generation or higher recommended)
    • Intel® Pentium® processors
    • Intel® Xeon® processors
    • Processors with Intel® 64 architecture
  • Intel® IPP installed as a standalone package or as part of one of the following suites:
    • Intel® Parallel Studio XE
    • Intel® System Studio
  • OpenGL* developer package (required only for building on Linux* OS)
  • X11* developer package (required only for building on Linux* OS)
