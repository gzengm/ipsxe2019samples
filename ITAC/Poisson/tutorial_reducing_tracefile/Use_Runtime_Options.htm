Tutorial: Reducing Trace File Size for Intel® Trace Analyzer and Collector

Use Runtime Options

 Use the -trace-collectives and -trace-pt2pt options to get information only about the Collective and Point-to-Point operations in the application.

  • Get Trace Data on Collective Operations

  • Get Trace Data on Point-to-Point Operations

Note

These options are only supported at runtime with the Hydra Process Manager.

Note

The –trace-collectives and –trace-pt2pt options use the configuration file specially set for these options and redefine the VT_CONFIG environment variable. When you use these options, Intel® Trace Collector will not read your own configuration file.
