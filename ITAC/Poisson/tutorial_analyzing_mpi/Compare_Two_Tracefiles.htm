Compare Two Trace Files

 Compare two trace files with the help of the Comparison View. To open a Comparison View for the original application trace file (poisson_sendrecv.single.stf), go to View > Compare. In the dialog that appears, choose the trace file of the revised application (poisson_icomm.single.stf). The Comparison View shows an Event Timeline for each trace file and a Comparison Function Profile chart.
