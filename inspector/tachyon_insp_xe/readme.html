Intel® Inspector Sample

Tachyon is a parallel raytracer / renderer used in combination with the Intel® Inspector tutorial.

This example includes software developed by John E. Stone. See here for copyright information.

Quick Start on Windows*
  1. Install Intel® Parallel Studio XE
  2. See tachyon_insp_xe\msvc\README.txt file. For more details, please use the tutorial from the Intel® Inspector product.
Quick Start on Linux*
  1. Install Intel® Parallel Studio XE
  2. Use the project as directed in the Intel® Inspector tutorial.
System Requirements:
  Check to expand...
    The sample should work on most systems. For details on what is required, please refer to the product release notes. The tutorial implementation uses Intel® Threading Building Blocks (Intel® TBB) library.
Build Instructions:
For Microsoft Visual Studio* users:   Check to expand...
    The samples package contains a Visual Studio solution named tachyon_samples.sln configured to use the Intel® C++ Compiler. The solution promotes common use cases for Visual Studio projects such as building, running, or debugging an application. Below are the instructions for key tasks which can be done with the samples.

    To Compile All of the Projects in the Solution

    • Click Build > Rebuild Solution
For Linux* command-line users:   Check to expand...
    The samples package contains a Makefile configured to use the Intel® C++ Compiler. The Makefile promotes common use cases such as building, running, or debugging an application.

    To Build the project:

  • Enable your particular compiler environment for the Intel® C++ Compiler.
  • In a terminal session, change directory to the tachyon_insp_xe directory.
  • Type make
