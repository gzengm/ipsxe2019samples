========================================================================
    Fortran Console Application : "WinPrint_Direct" Project Overview
========================================================================

Intel Visual Fortran sample project WinPrint_Direct

Demonstrates use of the Win32 APIs writing directly to a
Windows printer.

WinPrint_Direct.f90 has the module with the routine that does
the printing
Test_WinPrint_Direct.f90 is a test program
test.prn is a PostScript* sample file which, when printed,
shows This is some text".
