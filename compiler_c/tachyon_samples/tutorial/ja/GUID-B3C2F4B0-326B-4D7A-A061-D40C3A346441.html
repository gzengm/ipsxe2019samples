サンプル・アプリケーションを準備する

このチュートリアルでは、tachyon_samples.sln ソリューション・ファイルを使用します。ソリューションには、次のプロジェクトが含まれます。

  • build_serial
  • build_with_openmp
  • build_with_openmp_solution
  • build_with_tbb
  • build_with_tbb_solution
  • tachyon.common

注

プロジェクト build_with_openmp と build_with_tbb (および対応するソリューション) は、それぞれ OpenMP* とインテル® スレッディング・ビルディング・ブロック (インテル® TBB) を使用します。これらの実装に加えて、インテル® TBB に基づくラムダ関数も使用できます。

プラットフォームおよびその他の詳細について

このサンプル・アプリケーションのソリューションは、Microsoft* Visual Studio* 2012 で作成されました。Microsoft* Visual Studio* 2013 以降で tachyon_compiler.sln ソリューションを開くと、対応するバージョンの Microsoft* Visual Studio* ソリューションに変換されます。

すべての実装の実行ファイルとオブジェクトは、tachyon_samples\msvs\<project_name>_<arch>_Debug または tachyon_samples\msvs\<project_name>_<arch>_Release ディレクトリーに生成されます。
